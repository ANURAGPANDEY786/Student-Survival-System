# 🎓 Student Survival System (StudentBuddy)

A web-based platform designed to help students manage essential needs such as accommodation, food services, and safety information when moving to a new city for studies.

---

## 📌 Project Overview

The **Student Survival System (SSS)** is a platform built to assist students who relocate for education and struggle to find reliable information about accommodation, food services, and nearby facilities.

This system provides a centralized solution where students can explore **PG accommodations, food services, and student reviews** to make better decisions.

The goal of the project is to simplify the student experience by providing important survival resources in one place.

---

## 🚀 Key Features

### 🔐 Student Authentication
- Secure student login system
- Dashboard access after login

### 🏠 PG & Room Listings
- Browse nearby PG accommodations
- View rent details and facilities
- Access student reviews

### 🍽 Food & Mess Information
- Discover nearby food services
- Compare meal options for affordability

### ⚠ Safety Indicator System
Listings include safety indicators:

- 🟢 Safe  
- 🟡 Warning  
- 🔴 High Risk  

This helps students make safer decisions when selecting accommodation.

### 📝 Review System
Students can:
- Submit PG reviews
- Share experiences
- Rate accommodation quality
